현재 인스타 게시물만 다시 가져오기 때문에 인스타에서 삭제한 게시물은 다음 동기화 때 웹에서도 내려갑니다.

지금 동기화
  1. GitHub 저장소의 Actions Secret에 INSTAGRAM_ACCESS_TOKEN을 한 번 등록합니다.
  2. 연동 후 6시간마다 자동 갱신되며, 위 버튼에서 즉시 실행도 가능합니다.
  3. 캐러셀은 카드 안에 여러 장으로 들어가고 릴스는 대표 이미지와 원문 링크를 사용합니다.
