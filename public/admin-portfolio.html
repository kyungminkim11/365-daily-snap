운영 흐름

  1. 프로젝트 제목과 촬영 카테고리를 정합니다.
  2. 태그는 가능한 선택형으로 고르고, 새 표현만 직접 추가합니다.
  3. 위치는 프리셋을 고른 뒤 지도에서 실제 검색 결과를 확인합니다.
  4. 사진·영상을 여러 개 올리고 공개 여부를 저장합니다.
